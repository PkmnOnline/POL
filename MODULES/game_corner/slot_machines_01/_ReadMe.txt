Slot Machines
	This minigame operates on Coins. This version represents the very first slot machines from the Pokemon Green game.

	The machines can be played for 1, 2, or 3 coins. Paying 1 coin allows payouts only on the center line. Paying 2 allows payouts on all 3 horizontal lines. Paying 3 allows payouts on all 3 horizontal lines as well as the two diagonal lines. Each Reel consists of the following icons, which when lined up in three results in a payout values:
		7			300
		Bar			100
		Clefairy	15
		Gengar		15
		Voltorb		15
		Cherries	8
